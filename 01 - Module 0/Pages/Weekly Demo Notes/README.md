<p>Creating a Scene in Unreal Engine: Demo Notes for a First-Time Teacher</p>
<p>Introduction:</p>
<ul>
<li>Greet the students and introduce yourself.</li>
<li>Explain that today's demo will focus on creating a scene in Unreal Engine using Quixel assets.</li>
<li>Briefly explain the importance of creating visually appealing scenes in game development or other 3D applications.</li>
</ul>
<p>Part 1: Setting up Unreal Engine and Quixel Bridge</p>
<ol>
<li style="list-style-type: none;">
<ol>
<li>
<p>Launch Unreal Engine. a. If necessary, guide students through installing Unreal Engine. b. Create a new project or open an existing one.</p>
</li>
<li>
<p>Download and install Quixel Bridge. a. Explain that Quixel Bridge is a tool that integrates the Quixel Megascans library with Unreal Engine, making it easy to import high-quality assets. b. Guide students through the installation process, if needed.</p>
</li>
</ol>
</li>
</ol>
<p>Part 2: Importing Quixel Assets</p>
<ol>
<li style="list-style-type: none;">
<ol>
<li>
<p>Open Quixel Bridge and sign in using an Epic Games account. a. If students do not have an account, provide instructions on creating one.</p>
</li>
<li>
<p>Browse the Quixel Megascans library. a. Explain the different types of assets available (3D models, materials, textures, etc.). b. Encourage students to choose a few assets that they would like to use in their scene.</p>
</li>
<li>
<p>Download and export assets to the Unreal Engine project. a. Show students how to click the "Export" button to send the assets to their project. b. Explain that the assets will appear in the "Megascans" folder within the Content Browser in Unreal Engine.</p>
</li>
</ol>
</li>
</ol>
<p>Part 3: Creating a New Level</p>
<ol>
<li style="list-style-type: none;">
<ol>
<li>
<p>In Unreal Engine, go to "File" &gt; "New Level" to create a new level. a. Explain what a level is and its purpose in game development.</p>
</li>
<li>
<p>Choose a template for the level (e.g., "Default" or "Time of Day"). a. Briefly explain the differences between the templates and how they can affect the scene.</p>
</li>
<li>
<p>Save the level by clicking "File" &gt; "Save Current," and provide a name for the level.</p>
</li>
</ol>
</li>
</ol>
<p>Part 4: Placing Quixel Assets in the Scene</p>
<ol>
<li style="list-style-type: none;">
<ol>
<li>
<p>Open the Content Browser and navigate to the "Megascans" folder.</p>
</li>
<li>
<p>Locate the imported Quixel assets. a. Show students how to search and filter assets in the Content Browser.</p>
</li>
<li>
<p>Drag assets into the scene. a. Demonstrate how to place, move, rotate, and scale assets. b. Encourage students to experiment with asset placement to create their own unique scene.</p>
</li>
</ol>
</li>
</ol>
<p>Part 5: Tips for Creating an Interesting Scene</p>
<ol>
<li style="list-style-type: none;">
<ol>
<li>
<p>Discuss various techniques for creating visually appealing scenes: a. Composition: Rule of thirds, leading lines, framing, etc. b. Lighting: Different light sources, colors, and intensities. c. Color: Harmonious color palettes, complementary colors, analogous colors, monochromatic color schemes. d. Scale and Proportion: Varying sizes and proportions of objects. e. Detail and Texture: Adding high-quality details and textures using Quixel assets. f. Camera Angle and Perspective: Experimenting with angles, focal lengths, and depth of field settings.</p>
</li>
<li>
<p>Encourage students to apply these techniques to their own scenes.</p>
</li>
</ol>
</li>
</ol>
<p>Conclusion:</p>
<ul>
<li>Summarize the main points of the demo.</li>
<li>Remind students of the importance of creating visually appealing scenes in game development.</li>
<li>Invite questions and provide guidance on any unclear aspects.</li>
<li>Encourage students to continue practicing and exploring Unreal Engine and Quixel assets on their own.</li>
</ul>
