<p><strong><span style="font-size: 14pt;">Objective</span></strong></p>
<p><span>Students will learn how to apply the last stage of polish on their games using Unreal Engine. There are also videos about career advice and a video goes over the instructor's first game project. <strong>Students must also have a minimum of one person play their game and get feedback on the game.&nbsp;</strong></span></p>
<ol style="list-style-type: decimal;">
<li><span>Polishing and End Game</span></li>
<li><span>Career in Gaming</span></li>
<li><span>Go Over Instructor Project</span><strong></strong></li>
<li><strong>Playtesting </strong><strong></strong>Making a project from start to finish in Unreal Engine is no small task. However, making a game that is fun and engaging is an entirely different one. We have people playtest games because that is the end goal of the game. It is ultimately a product that others consume. When conducting a playtesting session, there are a few rules that are good to follow.&nbsp;</li>
</ol>
<p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;1. Keep as quiet as possible. This will help you to discover how intuitive your UI/UX is.&nbsp;&nbsp;</p>
<p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;2. Log all questions and feed back. If the person is trying to be helpful, you will receive honest feedback and a helpful discussion.</p>
<p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;3. Playtest with intent. Have some questions prepared to ask the player at the end of the session.</p>
