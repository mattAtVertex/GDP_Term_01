# Assignment 04: Modular Door and Key

<p>This assignment, we will create an opening door. The door need to open when the player overlaps the trigger and the door needs to be able to set to lock or unlock.&nbsp;</p>
<p>Also in this assignment, you will need to create a key pickup that can be used to unlock the door.&nbsp;</p>
<p>Once the door is unlocked, your door will open automatically upon overlap or by interaction.</p>
<p>Once the system is complete, you will submit a video of the door functioning as described above and you will also provide screenshots of your node networks.&nbsp;</p>
<p>You can upload your files to syncsketch and then submit a link here in this assignment.</p>
